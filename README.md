## API Endpoints (Postman Testing)
This project can be tested on **Postman**.

### Add a Book
- **Method**: POST  
- **URL**: `localhost:8080/book`

### Update a Book
- **Method**: PUT  
- **URL**: `localhost:8080/book/{id}`

### Delete a Book
- **Method**: DELETE  
- **URL**: `localhost:8080/book/{id}`

### Find a Book by ID
- **Method**: GET  
- **URL**: `localhost:8080/book/{id}`

### Find a Book by ID and Name
- **Method**: GET  
- **URL**: `localhost:8080/book/{id}/{name}`
